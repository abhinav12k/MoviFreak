# MoviFreak
A simple Android App which helps the user to discover movies and TvShows.It can be used to view reviews and trailers of latest and top rated movies or TvShows. It's a home for a movie/TvShow lover. Get everything you need to know about a movie/TvShow.

# Features:
* Discover the home for the most popular, top rated movies and TvShows
* Watch latest movie or TvShow trailers and teasers
* Read reviews from other users
* Material Design
* Day/Night Mode
* Mark Movies or TvShows as Favorites
* No need to leave the app to view Trailers and teasers(InApp Video Play)
* Save Movies or TvShows for offline Support
